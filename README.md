## 软件介绍 / Introduction

基于 [MaaFramwork[(https://github.com/MaaXYZ/MaaFramework) 框架的明日方舟 PC 端简易划火柴实现，本项目是 AUTO-MAS 的衍生项目。

当前已实现：

- 1 帧部署干员
- 1 帧选中干员
- 1 帧技能释放
- 1 帧撤退干员

> 1 帧的意思是每次操作耗费约 1 帧

默认键位：

| 按键 | 功能 |
| --- | --- |
| s | 任意状态按住待部署区干员不放，再按键拾取干员 |
| w | 鼠标悬浮或按住已部署干员，再按键选中干员 |
| r | 鼠标悬浮或按住已部署干员，再按键打开干员技能 |
| t | 鼠标悬浮或按住已部署干员，再按键撤退干员 |
